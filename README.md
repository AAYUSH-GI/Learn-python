# PyPath — Interactive Python Learning Platform

A modern React + TypeScript web app inspired by coddy.tech featuring:
- Visual roadmap with locked/unlocked lesson nodes
- 5 complete Python lessons with theory, examples, exercises, and quizzes
- In-browser Python execution via Pyodide
- Gamified XP/streak progress tracking with localStorage persistence
- Dashboard, leaderboard, profile, settings, auth placeholder, and 404 pages

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
